# ZB_CS1103_FinalProject
Final Project for CS 1103. Database containing elements from Formula 1: races, drivers, teams, etc.

USERS GUIDE
When the program starts, you will see this menu:

View championship standings
View races
View race result
Input race results
Disqualify a driver
Exit

Option 1:
Shows the driver standings sorted by points and also shows team standings.

Option 2:
Shows all races with their ID, name, and location.

Option 3:
Lets you enter a race ID and then shows the results for that race. If no results exist yet, it will say so.

Option 4:
Lets you input race results. You choose a race, then enter driver IDs for positions 1 through 10. Points are added automatically and standings are updated.

Option 5:
Lets you remove a driver from a race. You enter the race ID and driver ID, and the result is deleted. Standings are updated after.

Option 6:
Exits the program.

Notes:
Points are calculated automatically based on race results.
Each driver can only appear once per race.
Each position in a race can only be used once.
Option 5 removes the result, but does not shift positions upward.
